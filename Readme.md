# HParser
## 簡介
這是一個用來做 類型 與 字串 互轉的簡陋轉換器。   
```H``` 的意思是 ```Horrible```，它效能超爛，跟作者一樣一點也不優秀，純粹是用來做概念證明用。  
雖然沒有效能能讓你的專案常保笑容，但它可以做到大部分的轉換。
